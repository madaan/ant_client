    Project description

    This application communicates with a remote translator server running moses and fetches the translations.

    Beyond providing the translations, the application supports the following features:

    1. Text to speech -Listen to the foreign language sentence in varying speeds

    2. Caching of good translations -Good translations can be cached for quick reference

    3. Visualization of the translation process -Show the mapping of words from foreign language to the target language

    4. Scoring -Scores from moses are used to give the user a handle over the reliability of the translations
